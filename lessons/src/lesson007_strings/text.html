In Gleam Strings are written as text surrounded by double quotes.

They can span multiple lines and support unicode characters.

Several escape sequences are supported:

  • \" - double quote
  • \\ - backslash
  • \f - form feed
  • \n - newline
  • \r - carriage return
  • \t - tab
  • \u{1F600} - unicode codepoint

The gleam/string standard library module contains functions for working with strings.
